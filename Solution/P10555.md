## [P10555](https://www.luogu.com.cn/problem/P10555)

题意：给定一棵 $n$ 个节点的树，每个点有权值 $a_i$。以 $k$ 为根时，树被划分为 $\deg(k)$ 个连通块（$k$ 的每个邻居子树各自成块）。需要选出若干权值，每个块至少选一个，同一个点可被多次选择（第 $c$ 次代价为 $a_i+c-1$）。对每个 $k$ 输出最小总代价。

---

## 核心引理

> **引理**：确定每个块选的第一个最小值后，剩余选择等价于——将所有「已选的块最小值 $+1$」放回全局多重集，然后反复取全局最小，每取一个就将它的值 $+1$ 放回。

于是问题变为对每个 $k$：

- 知道 $\deg(k)$ 个块各自的最小值；
- 将这些最小值从全局多重集中取出、累加，$+1$ 后放回；
- 再进行若干轮「取全局最小 $\to$ 累加 $\to$ $+1$ 放回」。

关键就是快速获得「以 $k$ 为根时各块最小值」。这是经典的**换根 DP**。

---

## 换根 DP：子树 min 与父方向 min

### 第一遍 DFS：子树最小值

`w[u]` 表示以 $1$ 为根时 $u$ 子树的最小权值。同时记录使 `w[u]` 取最小的儿子 `miv` 和次小儿子 `miv2`——换根时用来 $O(1)$ 排除某个儿子。

```cpp
void Dfs1(int u,int fa) {
    w[u]=a[u];
    int miv=0,miv2=0,mi2=INF;
    for(int v:e[u]) if(v^fa) {
        Dfs1(v,u);
        if(w[u]>w[v]) 
            w[u]=w[v],miv=v;
    }
    for(int v:e[u]) if(v^fa&&v^miv) {
        if(mi2>w[v])
            mi2=w[v],miv2=v;
    }
    mi[u]={miv,miv2};
}
```

`w[0]=INF` 已在全局初始化，保证「没有儿子」时次小值自然为 INF。

### 第二遍 DFS：换根

`las` 是从父亲方向传下来的最小值（不含当前子树）。`k` 是儿子 $v$ 从父方向看到的最小值。

```cpp
void Dfs2(int u,int fa,int las) {
    las=min(las,a[u]);
    for(int v:e[u]) if(v^fa) {
        int k=min(las,w[v==mi[u].first?mi[u].second:mi[u].first]);
        p[u].push_back(w[v]);
        p[v].push_back(k);
        Dfs2(v,u,k);
    }
}
```

为什么 `v==mi[u].first ? mi[u].second : mi[u].first`？

- 要算「$u$ 除去子树 $v$ 后的最小值」。如果 $v$ 恰好是 $u$ 最小的儿子（`mi[u].first`），那就只能取次小儿子 `mi[u].second`；否则取最小儿子 `mi[u].first`。
- 再与 `las`（来自 $u$ 父亲方向的值）取 min，就是 $v$ 从父方向看到的完整最小值。

同时填充 `p[u]`：对 $u$ 的每个邻居方向，存入该方向的块最小值。以 $u$ 为根时，$p[u]$ 恰好包含所有块的最小值。

## 全局多重集与回滚栈

用 `map<int,int>` 做多重集（键=权值，值=出现次数），取最小直接 `mp.begin()->first`。由于不同 $k$ 的 `Calc` 相互独立，用栈记录修改后统一回滚。

```cpp
stack<pii>st;
void Add(int u,bool b){
    mp[u]++;
    if(b) st.push({u,1});
}
void Del(int u,bool b){
    mp[u]--;
    if(!mp[u]) 
        mp.erase(u);    // 计数归零时删除键，保证 begin() 有效
    if(b) st.push({u,-1});
}
int Query(){
    return (*mp.begin()).first;
}
```

参数 `b`：`1` 表示记录到回滚栈，`0` 表示不记录（回滚时用 `0` 避免死循环）。`mp.erase(u)` 在计数归零时清理键，保证 `mp.begin()` 始终有效。

## Calc 过程

```cpp
ll Calc(int u) {
    if(p[u].size()==1) 
        return 0;                        // (1)
    ll ans=0;
    Del(a[u],1);                         // (2)
    for(int v:p[u]) 
        ans+=v,Del(v,1),Add(v+1,1);      // (3)
    int d=p[u].size()-2;                 // (4)
    while(d--){
        int v=Query();
        ans+=v;
        Del(v,1),Add(v+1,1);             // (5)
    }
    while(st.size()){
        auto[t,w]=st.top();
        st.pop();
        if(w==1) 
            Del(t,0);
        else Add(t,0);                   // (6)
    }
    return ans;
}
```

### (1) 叶子特判

`p[u].size()==1` 即 $\deg(u)=1$，$u$ 是叶子。此时只有一个块，该块的最小值在 `Dfs2` 中已被邻居的 `Calc` 处理。对叶子自身返回 $0$。

### (2) 移除根自身

以 $u$ 为根时，$a_u$ 不属于任何块，不能参与选择。从全局多重集中删掉它。

### (3) 每块选最小值（第一轮贪心）

遍历 $p[u]$ 中 $\deg(u)$ 个块的最小值。对每个 $v$：
- `ans+=v`：累加；
- `Del(v,1)`：从多重集删掉该值（它被消耗了）；
- `Add(v+1,1)`：把 $v+1$ 放回（模拟该点被选一次后权值 $+1$，为后续选择做准备）。

这一步结束后，所有块都满足了「至少选一个」的约束。

### (4) 剩余轮数

`d = p[u].size()-2 = deg(u)-2`。第一轮已选了 $\deg(u)$ 个，还需选 $\deg(u)-2$ 个。总选择次数为 $2\deg(u)-2$。

### (5) 全局贪心（后续轮）

```cpp
while(d--){
    int v=Query();       // 取全局最小
    ans+=v;
    Del(v,1),Add(v+1,1); // 消耗当前值，+1 放回
}
```

每轮操作与 (3) 后半部分完全一致：取最小、累加、删除、$+1$ 放回。这正是引理中「全局贪心」的实现。

### (6) 回滚

```cpp
while(st.size()){
    auto[t,w]=st.top();
    st.pop();
    if(w==1) Del(t,0);  // 之前 Add 过的，现在 Del 回去
    else     Add(t,0);  // 之前 Del 过的，现在 Add 回去
}
```

栈中记录的是「操作前的逆操作」：`w=1` 表示当时做了 `Add`，回滚时做 `Del`；`w=-1` 表示当时做了 `Del`，回滚时做 `Add`。参数 `0` 表示不再次记录到栈中。回滚后多重集恢复到调用 `Calc(u)` 之前的状态，可以安全处理下一个 $u$。

## Code

```cpp

int n,m;
int a[maxn],w[maxn];
map<int,int> mp;
vector<int>e[maxn],p[maxn];
pii mi[maxn];

void Dfs1(int u,int fa) {
    w[u]=a[u];
    int miv=0,miv2=0,mi2=INF;
    for(int v:e[u]) if(v^fa) {
        Dfs1(v,u);
        if(w[u]>w[v]) 
            w[u]=w[v],miv=v;
    }
    for(int v:e[u]) if(v^fa&&v^miv) {
        if(mi2>w[v])
            mi2=w[v],miv2=v;
    }
    mi[u]={miv,miv2};
}
void Dfs2(int u,int fa,int las) {
    las=min(las,a[u]);
    for(int v:e[u]) if(v^fa) {
        int k=min(las,w[v==mi[u].first?mi[u].second:mi[u].first]);
        p[u].push_back(w[v]);
        p[v].push_back(k);
        Dfs2(v,u,k);
    }
}
stack<pii>st;
void Add(int u,bool b){
    mp[u]++;
    if(b) st.push({u,1});
}
void Del(int u,bool b){
    mp[u]--;
    if(!mp[u]) 
        mp.erase(u);
    if(b) st.push({u,-1});
}
int Query(){
    return (*mp.begin()).first;
}
ll Calc(int u) {
    if(p[u].size()==1) 
        return 0;
    ll ans=0;
    Del(a[u],1);
    for(int v:p[u]) 
        ans+=v,Del(v,1),Add(v+1,1);
    int d=p[u].size()-2;
    while(d--){
        int v=Query();
        ans+=v;
        Del(v,1),Add(v+1,1);
    }
    while(st.size()){
        auto[u,w]=st.top();
        st.pop();
        if(w==1) 
            Del(u,0);
        else Add(u,0);
    }
    return ans;
}

void solve(){
    Dfs1(1,0);
    Dfs2(1,0,INF);
    for(int u=1;u<=n;++u)
        cout<<Calc(u)<<" ";
}
void init(){
    cin>>n;
    w[0]=INF;
    for(int i=1;i<=n;++i) 
        cin>>a[i],mp[a[i]]++;
    int u,v;
    for(int i=1;i<n;++i) 
        cin>>u>>v,e[u].push_back(v),e[v].push_back(u);
}
```

## 复杂度

- `Dfs1 + Dfs2`：$O(n)$，每个点每条边访问常数次。
- `Calc`：每个 $u$ 操作 $O(\deg(u))$ 次 `map` 增删查（每次 $O(\log n)$）。$\sum \deg(u) = 2n-2$，所以所有 `Calc` 总计 $O(n\log n)$。
- 每个 $u$ 的回滚量也是 $O(\deg(u))$，均摊一致。

**总复杂度**：$O(n\log n)$，空间 $O(n)$。
